# Lecture 5 — Введение в командную строку (CLI)

**Тема:** «Основы работы в командной строке Linux/Windows»

**Цель урока:** Познакомиться с интерфейсом командной строки, научиться базовым командам для навигации и работы с файлами.

**Время:** ~20 мин.

**Ключевые понятия:** «CLI, терминал, командная оболочка (shell), bash, PowerShell, команды (ls, cd, mkdir, dir, pwd)»

---

### 1) Введение

Графический интерфейс (GUI) удобен для повседневных задач, но он скрывает многие детали работы ОС. Командная строка (CLI) — это более мощный и гибкий инструмент управления системой. Администраторы и разработчики часто используют команды для автоматизации задач, настройки серверов и отладки программ.

---

### 2) Основная часть

**Основные понятия:**

- **Терминал:** Программа, в которой вы вводите команды (CMD в Windows, Terminal в macOS/Linux).
- **Shell (оболочка):** Интерпретатор команд. В Linux это чаще всего **bash**, в Windows — **PowerShell** или **cmd**.
- **Команда:** Это либо встроенная функция оболочки, либо имя программы, которую нужно запустить.

**Базовая навигация (Linux/macOS):**

| Команда | Описание | Пример |
|---------|----------|--------|
| `pwd` | Показать текущую папку | `pwd` |
| `ls` | Показать содержимое папки | `ls -la` |
| `cd` | Сменить папку | `cd /home/user` |
| `mkdir` | Создать папку | `mkdir NewFolder` |
| `rm` | Удалить файл | `rm file.txt` |
| `cp` | Скопировать | `cp file1.txt file2.txt` |
| `mv` | Переместить или переименовать | `mv old.txt new.txt` |

**Базовая навигация (Windows cmd):**

| Команда | Описание | Пример |
|---------|----------|--------|
| `cd` | Показать или сменить папку | `cd C:\Users` |
| `dir` | Показать содержимое папки | `dir` |
| `mkdir` | Создать папку | `mkdir NewFolder` |
| `del` | Удалить файл | `del file.txt` |
| `copy` | Скопировать | `copy file1.txt file2.txt` |
| `move` | Переместить | `move file.txt D:\` |

### 3) Пример/практика

Создадим структуру проекта через командную строку:

```bash
# Перейдем в домашнюю папку
cd ~

# Создадим папку проекта
mkdir my_project

# Зайдем в нее
cd my_project

# Создадим вложенные папки
mkdir css js images

# Создадим пустой файл
touch index.html

# Посмотрим результат
ls -la
